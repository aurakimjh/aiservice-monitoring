## 1. Java Agent EOL 선언

AITOP은 **2026-09-30**을 기준으로 Java Agent(aitop-onsite)의 **일반 지원을 종료(EOL)**합니다.

```
Java Agent (aitop-onsite)
최종 릴리스 버전: v1.9.x (Maintenance Only)
EOL 날짜: 2026-09-30
```

EOL 이후에는 Critical 보안 취약점에 한해 **2027-03-31까지 유지보수 전용 패치**를 제공합니다.

## 7. FAQ

### Q. EOL 이후 Java Agent를 계속 사용하면 어떻게 되나요?
기존 설치된 Java Agent는 계속 동작합니다. 단, 신규 버그/보안 패치 및 기술지원이 제공되지 않습니다. 2027-03-31 이후에는 Critical CVE 패치도 제공되지 않으므로 보안 리스크에 노출될 수 있습니다.

### Q. EOL 이후 갑자기 Java Agent가 동작하지 않게 되나요?
아니요. EOL은 지원 종료를 의미하며, 원격에서 에이전트를 강제 종료하거나 기능을 제한하지 않습니다.

### Q. 특수 환경(AIX, 레거시 유닉스)은 어떻게 처리되나요?
개별 기술지원 협의를 통해 EOL 연장(최대 2026-12-31) 또는 맞춤 마이그레이션 방안을 제공합니다. `migration@aitop.io`로 문의하세요.

### Q. 마이그레이션 비용이 발생하나요?
병행 운영 기간(2026-06-25 이전) 내 전환 시 마이그레이션 지원은 무상입니다.
