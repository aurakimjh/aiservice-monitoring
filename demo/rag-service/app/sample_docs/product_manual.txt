SmartDesk Pro 사용자 가이드

제품 개요
SmartDesk Pro는 AI 기반 스마트 업무 관리 플랫폼입니다.
프로젝트 관리, 일정 조율, 문서 협업, 자동 보고서 생성 기능을 제공합니다.

1. 시작하기

1-1. 계정 생성
- 공식 웹사이트(smartdesk.example.com)에서 회원가입
- 이메일 인증 완료 후 로그인
- 초기 설정 마법사에서 팀 정보, 프로젝트 구조 설정

1-2. 워크스페이스 설정
- 대시보드 → 설정 → 워크스페이스에서 팀원 초대
- 역할 설정: 관리자, 편집자, 뷰어 3단계
- 프로젝트 폴더 구조는 최대 5단계 깊이까지 지원

2. 주요 기능

2-1. AI 일정 최적화
SmartDesk의 AI 엔진은 팀원들의 캘린더, 업무량, 우선순위를 분석하여
최적의 미팅 시간과 작업 일정을 자동으로 제안합니다.
- 캘린더 동기화: Google Calendar, Outlook 지원
- 자동 일정 제안: AI가 팀원 가용 시간을 분석하여 최적 시간 추천
- 충돌 감지: 일정 중복 시 자동 알림 및 대안 제시

2-2. 문서 협업
- 실시간 공동 편집 (최대 50명 동시 작업)
- 버전 관리: 모든 변경 이력 자동 저장, 원클릭 롤백
- 댓글 및 멘션: @팀원이름으로 알림 전송
- 템플릿: 회의록, 기획서, 보고서 등 20종 기본 제공

2-3. 자동 보고서 생성
AI가 프로젝트 진행 상황, 팀원 작업 현황, 이슈 목록을 분석하여
주간/월간 보고서를 자동으로 생성합니다.
- 자연어 요약: 복잡한 데이터를 이해하기 쉬운 문장으로 변환
- 차트 자동 생성: 진행률, 번다운 차트, 리소스 사용률
- PDF/PPT 내보내기: 프레젠테이션용 자동 포맷팅

3. 트러블슈팅

3-1. 동기화 오류 시
설정 → 계정 → 연결된 서비스에서 재연결을 클릭하세요.
문제가 지속되면 캐시를 삭제하고 브라우저를 새로고침하세요.

3-2. AI 기능이 작동하지 않을 때
AI 기능은 Pro 이상 플랜에서만 사용 가능합니다.
요금제를 확인하고, 필요 시 업그레이드하세요.

4. 요금제

- Free: 개인 사용자, 프로젝트 3개, AI 기능 제한
- Pro: 월 15,000원/인, 프로젝트 무제한, AI 기능 전체
- Enterprise: 별도 문의, SSO/SAML, 전용 지원, SLA 보장
